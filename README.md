# Nomads

**Nomads** — это лендинг на платформе WordPress, предназначенный для цифровых кочевников и удаленных работников. Шаблон включает адаптивный дизайн и простой процесс установки.

## Основные особенности:
- Адаптивный дизайн, подходящий для всех устройств.
- Легкая установка и настройка.
- Современный и чистый интерфейс.

## Установка:
1. Скачайте и разархивируйте проект.
2. Загрузите в папку `wp-content/themes/` вашего сайта WordPress.
3. Активируйте тему через панель управления WordPress.
